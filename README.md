# python-sec-tools
python编写的网络工具

1. 使用前的安装
   使用前先安装对应需要的package,使用pip命令在安装目录下输入指令：
   `pip install -r requirements.txt`
2. 使用注意事项
   使用的python版本为3.7.2
   使用时直接运行主文件mathin.py即可
   `python mathin.py`
